Code & Mates

Né d'une passion d'un des apprenants, le projet Codes & Mate consiste à présenter des profils ainsi qu'une formation de manière originale.

Mélangeant compétences techniques et qualités humaines, ce projet nous aura permis de faire nos premiers pas dans le développement web.

Travail en équipe, savoir-être et programmation, telles ont été nos ressources !
Issus d'environnements aussi divers que variés, nous avons en commun l'envie de réussir et un fort intérêt pour le développement web.

Notre Formation

Le développeur web est le responsable de la programmation d'un site web, il rédige et suit un cahier des charges pour créer le programme.
Il utilise des langages tels que JavaScript, Php, ou encore HTML, CSS.

Le Programme

Développer la partie visuelle d’un site ou d’une application web (front-end) :

  • Maquetter une application;
  • Développer une interface utilisateur web dynamique.

Développer les fonctionnalités d’un site, d’une application web (back-end) :

  • Créer une base de données;
  • Développer les composants d'accès aux données;
  • Développer la partie back-end d'une application web ou web mobile.
